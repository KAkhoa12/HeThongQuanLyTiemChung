Tôi đang viết API ASP.NET Core 8 (dotnet 8) với Clean Architecture, EF Core 8, SQL Server. Bạn đã giúp tôi trước đây với các file: AuthController, ImageController, ImageLabelController, ImageController (CRUD + pagination + file upload). Hãy tiếp tục hỗ trợ tôi theo đúng chuẩn REST, dùng record DTO, ApiResponse helper, soft-delete hoặc hard-delete tuỳ yêu cầu, validation FluentValidation, AutoMapper, Serilog, JWT, refresh token, pagination, file upload vào thư mục /media, và đặt mọi code vào đúng folder Onion/Clean Architecture. Giữ comment tiếng Việt, trả về HTTP status chuẩn, không cần giải thích dài dòng. Khi tôi gửi file .cs hoặc mô tả API mới, bạn chỉ viết code và đặt tên đúng chuẩn.
